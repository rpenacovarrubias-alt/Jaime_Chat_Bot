¿Cómo comparto este portal con un cliente?

Activar GitHub Pages

El portal ya está listo. Solo necesitas habilitarlo una vez desde la configuración del repositorio.

1
Abre la configuración del repositorio Ve a github.com/rpenacovarrubias-alt/Jaime_Chat_Bot → Settings → Pages
2
Selecciona el branch y carpeta En Source elige main branch → carpeta / (root) → clic en Save
3
Tu portal estará en línea en ~1 minuto URL pública: https://rpenacovarrubias-alt.github.io/Jaime_Chat_Bot/
4
Formulario de onboarding listo para compartir Envía este link al hotel cliente: https://rpenacovarrubias-alt.github.io/Jaime_Chat_Bot/plantillas/FORMULARIO_ONBOARDING_HOTEL.html
